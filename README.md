# 🍔 Swiggy Data Analysis (EDA)

This project focuses on **Exploratory Data Analysis (EDA)** of a Swiggy dataset using Python. The goal is to understand customer behavior, price distribution, and rating patterns through data cleaning and visualization.

---

## 📌 Project Objectives

* Analyze food delivery dataset
* Understand pricing trends
* Study customer ratings
* Identify outliers and patterns

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## 📊 Key Analysis Performed

* Data Cleaning (handling missing values)
* Outlier Detection
* Price Distribution Analysis
* Ratings Analysis
* Order/Review Trends
* Data Visualization

---

## 📈 Key Insights

* Most ratings are between 4.0 - 4.5 ⭐
* Price data contains outliers (very high values)
* Majority of orders are concentrated in low to mid range
* Customer behavior shows consistent rating patterns

---

## 📂 Project Structure

* `swiggy.ipynb` → Main analysis notebook
* `dataset.csv` → Dataset used (if added)

---

## 🚀 How to Run

1. Clone the repository
2. Open Jupyter Notebook
3. Run `swiggy.ipynb`

---

## 🎯 Conclusion

This project helps in understanding real-world data analysis and extracting meaningful insights using Python.
